Advantage Of Event Delegation

One Important thing to note, It uses event bubbling phase to handle event.

1. Less Code since, we just need to write event for parent

2. Can be used for Dynamic content such as infinite scrolling

3. Performance Optimization, since it will consume less memory

Disadvantage or Cons

1. Can't be applied for All events such as onfous, onblur, etc
